Lab 6: Deploying a Full-Stack Professional Portfolio
This lab focuses on developing and deploying a responsive personal portfolio using React. The application is built as a Single Page Application (SPA) with multiple routes, a contact form with validation and persistence, and dynamic components. The project demonstrates front-end development, local storage handling, and deployment to a cloud platform.
